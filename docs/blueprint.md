# **App Name**: Africa Heal Health

## Core Features:

- Homepage Showcase: Display a clean, intuitive home page with featured products and health information, using glassmorphic cards with light/dark mode adaptation.
- Product Catalog: Enable users to browse and filter products, and view detailed information, using glassmorphic cards with light/dark mode adaptation.
- Shopping Cart and Checkout: Allow users to add items to a cart and proceed through a simple checkout process, using glassmorphic cards with light/dark mode adaptation.
- AI Health Assistant: Offer an AI-powered health assistant for answering questions and suggesting products. This AI tool will decide, based on the content of a patient's question, when and if to provide suggestions.
- Intuitive Navigation: Implement a user-friendly interface with easy access to key features and information.
- Account and Theme: Support theme switching (light/dark mode) and user account management.
- Subscription Benefits Display: Highlight offers or discounts for subscribed customers, using glassmorphic cards with light/dark mode adaptation.

## Style Guidelines:

- Primary color: Mint green (#98FF98) to evoke health and freshness.
- Background color: Soft gray (#E0E0E0) for a modern look that suggests a frosted glass panel.
- Accent color: Soft white (#F8F8FF) to provide subtle contrast in glows and highlights.
- Headlines: 'Poppins' (sans-serif) for all major titles to keep a fresh and contemporary feel.
- Body Text: 'PT Sans' (sans-serif) for paragraphs, descriptions, and general text content.
- Use simple, clean, and consistent icons from the 'lucide-react' library.
- Incorporate glassmorphism using semi-transparent backgrounds, backdrop blur, rounded corners, and soft shadows.
